### 项目亮点
- 单节点硬件成本 **115-200元**（含太阳能供电）
- 30天真实大棚测试（瓦房店黄瓜大棚）：在线率97%、传输准确率98%、节水32%、增产16%
- 低成本、即插即用、适合中小农户
- 完整前后端代码 + 数据库 + 部署说明

### 技术栈
- **硬件**：ESP32 + 电容式土壤湿度传感器 + DS18B20 + 太阳能供电
- **通信**：MQTT + 阿里云IoT平台
- **后端**：Flask + MySQL + Redis
- **前端**：ECharts + HTML5 + JavaScript
- **数据清洗**：pandas定时任务

### 快速启动（本地）
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 初始化数据库
mysql -u root -p < init_db.sql

# 3. 启动Redis
redis-server --daemonize yes

# 4. 启动平台
python app.py
